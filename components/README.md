# Components

## SplashScreen.tsx
Professional animated splash screen with:
- Fade-in animation
- Spring-based scale effect
- Slide-up text animation
- Dark mode support
- Responsive design

## Logo.tsx
Reusable logo component that can be used throughout the app with automatic dark mode support.

## Usage

The splash screen is automatically displayed when the app launches. See `SPLASH_SCREEN_GUIDE.md` for detailed customization options.
